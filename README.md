# Data Cleaning Tool (Python GUI)

This is a Python-based Data Cleaning Tool with a graphical interface built using Tkinter.

## Features
- Upload CSV file
- Remove duplicates
- Handle missing values
- Visualize data (before & after cleaning)
- Download/Open cleaned dataset

## Tech Used
- Python
- Pandas
- Matplotlib
- Tkinter

## How to Run
1. Install required libraries:
   pip install pandas matplotlib

2. Run the app:
   python app.py

## Output
- Cleaned CSV file generated
- Graph visualization of missing data
